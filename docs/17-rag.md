# Retrieval-Augmented Generation (RAG)

## Overview

Kzle uses RAG to provide accurate answers based on uploaded documents instead of relying only on model knowledge.

---

## Workflow

Upload File

↓

Extract Text

↓

Chunk Text

↓

Embedding

↓

Store Embeddings

↓

Search

↓

LLM

↓

Response

---

## Advantages

Higher Accuracy

Reduced Hallucinations

Context Awareness

Lower Token Usage
