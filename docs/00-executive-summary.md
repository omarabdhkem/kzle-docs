# Executive Summary

Kzle is an AI Learning Workspace that helps learners organize knowledge, chat with AI, generate notes, summaries, flashcards, and quizzes inside dedicated workspaces.

## Vision
Build the best AI-powered learning platform.

## MVP
- Authentication
- Workspaces
- AI Chat
- File Uploads
- Notes
- Summaries
- Flashcards
- Quizzes
- Progress Tracking
