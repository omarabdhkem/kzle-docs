# User Journeys

## Journey 1: New Student
1. Sign up
2. Create a workspace
3. Upload a file or paste notes
4. Generate summaries, flashcards, and quizzes
5. Study and track progress

## Journey 2: Returning Student
1. Sign in from any device
2. Open an existing workspace
3. Continue studying from saved content
4. Review weak areas using quizzes and flashcards

## Journey 3: Teacher
1. Sign up as a teacher
2. Create a teacher profile
3. Publish learning material
4. Manage students and content
5. Offer subscriptions or course access
