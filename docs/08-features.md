# Features

## Workspace Management
Users can create unlimited workspaces for any subject, course, or goal.

## File Uploads
Users can upload learning materials such as PDFs and documents.

## AI Chat
Users can ask questions inside each workspace and receive context-aware answers.

## AI Summaries
The platform can turn notes or files into concise summaries and key points.

## Flashcards
The system can generate flashcards from learning content.

## Quizzes
The system can generate practice quizzes and review questions.

## Progress Tracking
Users can track study activity and review progress over time.

## Responsive Experience
The platform should work smoothly on mobile and desktop.

## Arabic and English Support
The platform should support both languages.

## Light and Dark Mode
Users should be able to choose a visual theme.
