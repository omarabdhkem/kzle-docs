# Product Overview

Kzle is an AI Learning Workspace designed for learners of all ages and levels. A user can create a workspace for any subject, upload learning materials, ask questions, generate summaries, create flashcards, and practice with quizzes.

## Key Concepts

- **Workspace**: A dedicated learning space for one subject, course, or goal.
- **AI Chat**: A context-aware assistant for questions related to the workspace.
- **AI Notes**: Automatic summaries and explanations from learning materials.
- **Flashcards**: Fast review cards generated from notes or files.
- **Quiz Generator**: Practice questions created from workspace content.
