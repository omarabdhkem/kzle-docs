# AI Architecture

Kzle uses AI to transform raw learning content into usable study material.

## AI Capabilities
- Summarize files and notes
- Generate flashcards
- Generate quizzes
- Answer questions in context
- Explain difficult concepts in simpler language

## Core Design
The AI system should work inside a workspace so that responses stay relevant to the selected subject or file set.

## Future Expansion
- Personalized tutoring
- Adaptive difficulty
- Study recommendations
- Smart revision planning
