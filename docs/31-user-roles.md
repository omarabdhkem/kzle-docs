# User Roles

## Overview

Kzle supports multiple user roles with different permissions and responsibilities.

---

## Roles

### Student

- Join workspaces
- Upload files
- Take quizzes
- Review flashcards
- Chat with AI
- Track progress

---

### Teacher

- Create learning content
- Manage classrooms
- Assign quizzes
- Review student progress
- Share notes
- Manage workspaces

---

### Organization Admin

- Manage organization
- Invite teachers
- Manage students
- View analytics
- Billing management

---

### Platform Admin

- Manage users
- Moderate content
- View system analytics
- Manage subscriptions
- Platform configuration

---
