# MVP

The MVP of Kzle should prove the core learning workflow with the smallest possible feature set.

## Goals
- Let a user sign up and sign in.
- Let a user create a workspace for any subject.
- Let a user upload notes or files.
- Let the system generate summaries, flashcards, and quizzes.
- Let the user ask AI questions inside the workspace.
- Keep everything synced across devices.

## MVP Feature Set
- Authentication
- Workspace creation
- File upload
- AI chat
- AI summaries
- Flashcard generation
- Quiz generation
- Basic progress tracking
- Arabic and English support
- Responsive UI
- Light and dark mode
